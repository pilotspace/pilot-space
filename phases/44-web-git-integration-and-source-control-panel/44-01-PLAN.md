   - `GitLabGitProvider(GitProvider)` stub (full implementation in Task 2)

   - `detect_provider(remote_url: str) -> str` that returns "github" or "gitlab" based on URL pattern

2. Extend `GitHubClient` in `client.py` with new methods (add after existing methods):
   - `get_ref(owner, repo, branch)` -- GET /repos/{owner}/{repo}/git/refs/heads/{branch}
   - `create_blob(owner, repo, content, encoding="utf-8")` -- POST /repos/{owner}/{repo}/git/blobs, returns SHA
   - `create_tree(owner, repo, base_tree, tree_entries)` -- POST /repos/{owner}/{repo}/git/trees, returns SHA
   - `create_git_commit(owner, repo, message, tree_sha, parent_shas)` -- POST /repos/{owner}/{repo}/git/commits, returns SHA
   - `update_ref(owner, repo, branch, sha)` -- PATCH /repos/{owner}/{repo}/git/refs/heads/{branch}
   - `compare_commits(owner, repo, base, head)` -- GET /repos/{owner}/{repo}/compare/{base}...{head}
   - `get_file_content(owner, repo, path, ref)` -- GET /repos/{owner}/{repo}/contents/{path}?ref={ref}
   - `list_branches(owner, repo, page=1, per_page=30)` -- GET /repos/{owner}/{repo}/branches
   - `create_branch(owner, repo, name, from_sha)` -- POST /repos/{owner}/{repo}/git/refs (creates refs/heads/{name})
   - `delete_branch(owner, repo, name)` -- DELETE /repos/{owner}/{repo}/git/refs/heads/{name}
   - `create_pull_request(owner, repo, title, body, head, base, draft=False)` -- POST /repos/{owner}/{repo}/pulls
   - `get_repo_info(owner, repo)` -- GET /repos/{owner}/{repo} (for default branch)

3. Add to `models.py`: `GitBlob(sha)`, `GitTreeEntry(path, mode, type, sha)`, `GitRef(ref, sha)`, `GitCompareResult(files, ahead_by, behind_by, total_commits)`

Multi-file commit flow in GitHubGitProvider.create_commit:
  a. get_ref to get HEAD SHA
  b. Get commit to get tree SHA (GET /repos/{owner}/{repo}/git/commits/{sha})
  c. create_blob for each file
  d. create_tree with all blob SHAs + base_tree
  e. create_git_commit with new tree + parent
  f. update_ref to point branch at new commit
  g. Handle 422 on update_ref (stale ref) by retrying once

Default tree mode: "100644" for all files (standard file permission).
