       - `useUploadPlugin(workspaceId: string)` — useMutation wrapping `uploadPlugin`, invalidates `['workspace-plugins']` on success, shows toast on success/error
       - `useTogglePlugin(workspaceId: string)` — useMutation wrapping `togglePlugin`, optimistic update on the query cache (toggle status in-place), rollback on error
       - `useDeletePlugin(workspaceId: string)` — useMutation wrapping `deletePlugin`, invalidates on success, shows confirmation toast
