1. Create TanStack Query hooks:

   `useGitStatus.ts`:
   - `useGitStatus(repo: GitRepo | null, branch: string)` -- uses `useQuery` with key `['git-status', owner, repo, branch]`
   - Calls `getRepoStatus`, passes `defaultBranch` as `baseRef`
   - On success, calls `gitWebStore.setChangedFiles(data.files)`
   - Enabled only when repo and branch are set
   - refetchInterval: 30000 (30s polling for live updates)

   `useBranches.ts`:
   - `useBranches(repo: GitRepo | null, search?: string)` -- uses `useQuery` with key `['git-branches', owner, repo, search]`
   - Calls `listBranches` with search param
   - Returns `{ branches, isLoading }`
   - Enabled only when repo is set

   `useCommit.ts`:
   - `useCommit(repo: GitRepo | null)` -- uses `useMutation`
   - Calls `createCommit` with staged files' content (fetched via `getFileContent`)
   - On success: invalidates `['git-status']` query, calls `gitWebStore.clearAfterCommit()`, shows success toast
   - On error: shows error toast with provider error message
   - Returns `{ commit, isCommitting }`

2. Create subcomponents:

   `ChangedFileItem.tsx`:
   - Props: `file: ChangedFile`, `onToggleStage: (path) => void`, `onSelect: (path) => void`, `isSelected: boolean`
   - Renders: checkbox (staged state), status icon (M=orange circle, A=green plus, D=red minus, R=blue arrow using Lucide icons), file path (basename bold, directory in muted), click to select
   - Checkbox click calls onToggleStage, row click calls onSelect
   - Selected file has accent background
   - NOT wrapped in observer() -- receives data via props

   `ChangedFileList.tsx`:
   - Props: `title: string`, `files: ChangedFile[]`, `onToggleStage`, `onStageAll?`, `onUnstageAll?`, `onSelect`, `selectedPath`
   - Uses Radix Collapsible -- section header with file count badge, expand/collapse
   - Header has "Stage All" or "Unstage All" button (+ icon / - icon)
   - Maps files to ChangedFileItem
   - Empty state: "No changes" text

   `CommitPanel.tsx`:
   - observer() component -- reads gitWebStore.commitMessage, gitWebStore.canCommit, gitWebStore.isCommitting
   - Textarea for commit message (placeholder: "Message (press Ctrl+Enter to commit)")
   - Commit button: disabled when !canCommit, loading spinner when isCommitting
   - Ctrl+Enter keyboard shortcut to commit
   - Calls useCommit mutation on submit

   `BranchSelector.tsx`:
   - observer() component -- reads gitWebStore.currentBranch
   - Popover + Command (cmdk) pattern (same as Phase 33 BranchSelector)
   - Shows current branch name with git-branch Lucide icon
   - Command input for search (debounced 300ms)
   - Branch list from useBranches hook
   - Default branch marked with star icon
   - Protected branches marked with lock icon
   - Click to switch: calls gitWebStore.switchBranch(name)
   - "Create Branch" item at bottom with inline input
   - Delete button on non-default, non-protected branches (with confirmation dialog)
  </action>
