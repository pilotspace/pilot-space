## Cross-Phase Integration

### Connected Exports (20+ verified)

All critical cross-phase connections wired:
- PilotSpaceAgent loads remote MCP servers (Phase 14 → Agent)
- PilotSpaceAgent routes model override (Phase 13 → Agent)
- PilotSpaceAgent materializes role skills (Phase 16 → Agent) + plugin skills (Phase 19 → Agent)
- MCP servers enforce DB-backed approval (Phase 18 → Phase 14 MCP handlers)
- Plugin install auto-creates action buttons (Phase 19 → Phase 17 SkillActionButtonRepository)
- Plugin uninstall deactivates action buttons (Phase 19 → Phase 17)
- SeedPluginsService fires on workspace creation (Phase 19 → Phase 12)
- Materializer reads from new tables with fallback (Phase 20 → Phase 16 legacy tables)
- SeedTemplatesService copies RoleTemplate rows to skill_templates (Phase 20 → Phase 16 data)
- Template catalog "Use This" → AI personalization via GenerateRoleSkillService (Phase 20 → Phase 16 service)
