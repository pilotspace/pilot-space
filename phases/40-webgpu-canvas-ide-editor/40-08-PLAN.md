**1. Create `frontend/src/features/editor/utils/jsonContentToMarkdown.ts`:**

Create a utility that converts TipTap JSONContent to a markdown string. The function walks the JSONContent tree and produces plain markdown text. This is needed because the note API returns `content` as JSONContent but MonacoNoteEditor expects a markdown string.

Implementation approach — use a recursive tree walker:
- `doc` node: join children with newlines
- `heading` node: prefix with `#` repeated by `attrs.level`, then inline text
- `paragraph` node: inline text of children
- `bulletList` / `orderedList`: prefix items with `- ` or `1. `
- `listItem`: render children as paragraphs
- `codeBlock`: wrap in triple backticks with optional `attrs.language`
- `blockquote`: prefix lines with `> `
- `horizontalRule`: `---`
- `text` node: apply marks — `bold` wraps in `**`, `italic` in `*`, `code` in backtick, `link` in `[text](href)`, `strike` in `~~`
- `hardBreak`: `\n`
- Unknown nodes: fallback to extracting text content recursively

Export: `export function jsonContentToMarkdown(content: JSONContent | undefined): string`

If content is undefined or null, return empty string.
