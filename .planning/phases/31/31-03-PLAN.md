    2. Add method after `get_by_workspace_and_id`:
       ```python
       async def count_active_by_workspace(self, workspace_id: UUID) -> int:
           """Count non-deleted MCP servers for a workspace.

           Used by the registration endpoint to enforce the per-workspace server cap.

           Args:
               workspace_id: The workspace UUID.

           Returns:
               Count of non-deleted MCP servers.
           """
           query = select(func.count()).where(
               and_(
                   WorkspaceMcpServer.workspace_id == workspace_id,
                   WorkspaceMcpServer.is_deleted == False,  # noqa: E712
               )
           )
           result = await self.session.execute(query)
           return result.scalar_one()
       ```

    **Router** (`workspace_mcp_servers.py`):
    1. Add constant at module level (after imports, before router definition):
       `MCP_SERVER_CAP = 10`
    2. In `register_mcp_server`, after the `await set_rls_context(...)` call and BEFORE the lazy imports / `WorkspaceMcpServer(...)` constructor, add:

       from pilot_space.infrastructure.database.repositories.workspace_mcp_server_repository import (

       repo = WorkspaceMcpServerRepository(session=session)

       if count >= MCP_SERVER_CAP:
